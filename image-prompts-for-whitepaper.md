## Usage Instructions

1. Copy each prompt individually
2. Use with your preferred AI image generator (Midjourney, DALL-E, Stable Diffusion, etc.)
3. Save generated images with filenames: `image1.png`, `image2.png`, etc.
4. Replace the `[IMAGE X: ...]` placeholders in the HTML white paper with actual `<img>` tags
5. Place images in the same directory as the HTML file or create an `images/` subdirectory

## Image Specifications

- **Format:** PNG or JPG
- **Resolution:** 1920x1080 or higher
- **Aspect Ratio:** 16:9 for diagrams, flexible for artistic visualizations
- **File Size:** Optimize for web (< 500KB per image)
- **Color Profile:** sRGB for web display
